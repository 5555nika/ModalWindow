# Modal Window App

A React application demonstrating asynchronous data fetching and modal window management.

## Features

- Data Fetching: Retrieves a user list via fetch witch a simulated network delay.
- State Management: handles loading, error, and success states effectively.
- Modal Interaction: Displays detailed user information in a modal window upon selection.
- TypeScript: Fully typed for better code maintainability.

  ## Tech Stack

  - React
  - TypeScript
